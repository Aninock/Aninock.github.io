复习中踩的坑

2019-12-27

C++

快要大一上期末考试了

比起实验 我更加怕笔试题 可能这就是IDE的力量吧

刷题完善了许多以前 不甚解的知识点：

1. C++程序设计步骤 及其 后缀名

1.编写:.cpp(.h) 2.编译:.obj 3.链接:.exe

2. C++程序基本单位

函数

3. 移植 与 兼容

程序:移植 硬件:兼容

4. 小数 默认值

double (注意形参为float的函数 不能直接调用 例如7.5之类的小数)

5. #define PI 3.14

PI代替的是字符串

6. 函数与全局变量的默认储存种类

extern

7. strlen 与 sizeof 对于字符串

strlen: 到 \0 停止 \0 不计入个数

sizeof: 到结尾才停止 结尾的\0 要加上

8. 数组做形参

退化为指针 大小变为4

9. [ ] 与 *

[ ] 结合优先级 比*高

10. 字符数组输出数组名

char s[ ]="12345";

cout<<s; 输出12345

cout<<s+2; 输出 345

11. 类的基本变量 能不能定义一个初值

不能

12. 初始化顺序

虚基类->非虚基类->派生类

13. if ...else... 是几条语句

一条

14. 派生类能不能改变基类的私有成员

不能 访问 都不能

15. 转义符号后面跟数字

一般是八进制

\x 后面跟十六进制

16. if(!x)等价于什么

if(!x)等价于if(x==0)

17. cin.get用法

如果 int a;

a=cin.get; 正确 cin.get(a); 错误

如果 char a;

a=cin.get; 错误 cin.get(a); 正确

注意：输入后 缓冲区会保留一个空格

18. 逻辑判断符号 && ||

一旦确定 后面直接省略

19. 函数定义能嵌套吗

不能 调用可以

20. 数组定义省略

int [a] [ ]错误 int [ ] [a] 正确

21. 字符串赋值

char a[20]="12345"; 正确

char *a; a="12345"; 正确

char a[20]; a="12345"; 错误

22. 重载函数的重载和返回值有关系吗

没关系

记载思路 若发现错误 请加QQ 交流
